🚀 Comment Utiliser les Statistiques Interactives

1. Accéder au Dashboard

Connectez-vous en tant qu'admin et cliquez sur le bouton "Statistiques Interactives" dans le header du dashboard.

2. Choisir les Statistiques

Sélectionnez le type de données à visualiser : Applications, Programmes, Utilisateurs, Tests, Chat, Financier ou Temporel.

3. Changer le Type de Graphique

Utilisez les boutons de type de graphique pour passer des barres aux lignes, camembert ou aires selon vos préférences.

4. Analyser les Données

Consultez les statistiques détaillées sous le graphique et les informations supplémentaires pour chaque type de données.
